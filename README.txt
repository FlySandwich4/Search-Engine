Step 1: Download the TK library

Since the openlab does not have the tk library and we are using the tk to build GUI, follow the below step to install the latest Tkinter. 

If you are using MacOs System:

Step A: Install the latest Python3
Step B: pip3 install tk


If you are using Windows System:

Step A: Install the latest Python3
Step B: pip3 install tk

==============================================================================================

Step 2: Build the inverted index

Step A: Open a new terminal
Step B: command:     python3 Index.py

==============================================================================================

Step 3: Run GUI

Step A: command:     python3 GUI.py

==============================================================================================

Step 3(Alternative): In case you can’t import tk

Step A: command:     python3 Query.py
Step B: command:     input the query
Step C: command:     input the number of results you want to have

==============================================================================================

Step 4(If you can open GUI): Type your query for search

Step A: Type in the query you want to search in the query box
Step B: Select the number of results you want to see
Step C: Click on the button “SEARCH IT!”
